Errors, Exceptions, and Warnings

  • System error
  • Programmer error
  • User error

Programmer errors are our responsibility, so we must ensure they are minimized as much as possible and fixed promptly

It might even be argued that user errors are in fact also programmer errors, because the program should be designed in a way that prevents the user from making the error.

Stack Traces

A stack trace will work backwards from the point at which the error occurred to identify the original function or method that started the sequence.

Strict Mode

Strict mode encourages a better quality of JavaScript to be written that befits a ninja programmer, so its use is recommended.

Linting Tools
